# Tecnicos Ya - Setup Script

Este script facilita la instalación completa de la aplicación (backend + frontend).

## 🚀 Uso Rápido

### Windows (PowerShell)
```powershell
python setup.py
```

### Mac/Linux (Bash)
```bash
python setup.py
```

## 📋 Lo que hace

1. ✅ Verifica Python 3.10+
2. ✅ Crea entorno virtual backend
3. ✅ Instala dependencias Python
4. ✅ Inicializa base de datos
5. ✅ Instala dependencias frontend (npm)
6. ✅ Genera archivo .env.example
7. ✅ Crea carpetas necesarias
8. ✅ Ejecuta tests (opcional)

## 📌 Requisitos Previos

- Python 3.10+
- Node.js 16+
- npm o yarn
- Git
